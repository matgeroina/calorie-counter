# Calorie Counter

A lightweight, responsive calorie tracking web application built with React.

## Live Demo
https://matgeroina.github.io/calorie-counter/

## Features
- Add and remove daily food entries  
- Set and persist calorie goals  
- Automatic daily totals and remaining calories  
- 7-day statistics overview  
- Data persistence via localStorage  
- Mobile-first responsive layout  

## Tech Stack
- React
- Vite
- React Router
- JavaScript (ES6+)
- CSS (Flexbox, Grid)
- GitHub Pages

## Getting Started
```bash
git clone https://github.com/matgeroina/calorie-counter.git
cd calorie-counter
npm install
npm run dev
